# Code of Conduct

TorchJD follows the [Linux Foundation Code of Conduct](https://lfprojects.org/policies/code-of-conduct/).

## Project Ownership

The TorchJD project is the property of SimplexLab. SimplexLab has full authority over the project and its governance, while maintainers are responsible for day-to-day operations and technical decisions. Maintainers are typically members of SimplexLab.

## Changes to this Code of Conduct

Changes to this Code of Conduct can only be made upon request from SimplexLab, which defines when and how such changes are possible.
